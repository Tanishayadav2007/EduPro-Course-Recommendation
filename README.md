# Student Segmentation and Personalized Course Recommendation System

## Project Overview
This project recommends personalized learning paths by analyzing student demographics, learning behavior, and academic performance.

## Features
- Data Preprocessing
- Missing Value Handling
- Exploratory Data Analysis
- Student Segmentation
- Personalized Course Recommendation
- Data Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset contains:
- User Information
- Course Details
- Enrollment Records
- Assessment Scores

## Workflow
1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Student Segmentation
6. Recommendation Generation
7. Visualization

## Results
The system groups students based on learning patterns and recommends suitable courses for improving their learning journey.
